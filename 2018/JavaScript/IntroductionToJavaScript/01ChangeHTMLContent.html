What can JavaScript do?

One of the things that I've heard this semester is that people don't know what they can do with Java Script. Fair point. This term, we are going to work on how to apply what we have just learned with JavaScript in Web pages.

One of the coolest parts of JavaScript is it's ability to change HTML Attributes.

What are attributes?

Good question, I'm glad you asked.

  • All HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are aalways specified in the starting tag
  • Attributes usually come in name/value pairs like: name="value"
